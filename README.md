# Abstract
  Influenza-like illness (ILI) remains a significant seasonal public health burden, and accurate forecasting is essential for preparedness and response. However, influenza transmission patterns have changed following the COVID-19 pandemic, limiting the performance of models that rely solely on meteorological variables. Wastewater-based epidemiology has emerged as a surveillance tool, yet its quantitative contribution to influenza prediction has not been fully evaluated in Korea.
  This study investigates whether incorporating a wastewater viral load indicator improves ILI forecasting using nationwide weekly data from the Republic of Korea (2023–2024). We develop deep learning models, including long short-term memory (LSTM) and gated recurrent unit (GRU) architectures, and compare performance across input configurations: temperature only, wastewater viral load only, and both. To address measurement uncertainty in the input data, we introduce a data augmentation framework combining jittering and Scaling. Augmentation parameters are determined through a scoring-function-based strategy, enabling construction of prediction intervals and robustness evaluation using the weighted interval score (WIS).
  The results demonstrate that integrating a wastewater-based indicator with meteorological variables improves predictive accuracy compared to models using meteorological inputs alone. Furthermore, the proposed augmentation framework enhances forecast stability and reduces uncertainty in prediction intervals. Among the tested architectures, the LSTM model showed superior performance.
  These findings provide empirical evidence that wastewater-based indicators offer incremental predictive value for ILI forecasting in Korea and highlight the importance of uncertainty-aware modeling strategies when incorporating environmental surveillance data. This study contributes to the development of robust influenza forecasting frameworks integrating wastewater-based epidemiology with deep learning models.
